# **App Name**: Lingua Blog

## Core Features:

- Bilingual Content Display: Display blog content in both Bangla and English, toggled by the user. Utilizes i18n support for seamless language switching.
- Language Toggle: Implement a language switch button for real-time content translation.
- Category Filtering: Enable users to filter blog posts by category (Tech, Education, Programming, Lifestyle, etc.).
- Search Functionality: Allow users to search for blog posts based on keywords. This tool improves content accessibility.
- Comment Section: Implement a comment section UI to allow users to post and view comments on blog posts.
- Social Sharing: Integrate social share buttons for users to share blog posts on social media platforms.
- Contact Form Submission: Enable users to send messages. No database.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5), reflecting intellect and sophistication.
- Background color: Light grayish-blue (#E8EAF6).
- Accent color: A muted purple (#7E57C2).
- Headline font: 'Poppins' sans-serif font for modern headlines; body font: 'PT Sans' sans-serif font for readability
- Note: currently only Google Fonts are supported.
- Use clean, minimalist icons from a library like Font Awesome or Material Icons for categories and UI elements.
- Responsive grid layout with a focus on readability and visual hierarchy.
- Subtle transition animations for page loading and UI interactions to enhance user experience.

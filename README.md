# ALIF Blog

ALIF Blog is a modern, responsive, and bilingual (Bangla & English) blogging platform designed for technology enthusiasts, educators, and lifelong learners. Built with performance and user experience in mind, it provides a seamless way to share and consume insightful articles.

## 🚀 Features

- **Bilingual Interface:** Toggle between English and Bangla seamlessly.
- **Dynamic Content Management:** Fully functional Admin Dashboard to manage posts, categories, comments, and subscribers.
- **Interactive Feedback:** Real-time comment system for every blog post.
- **Newsletter Subscription:** Integrated newsletter system with duplicate prevention and automated Telegram alerts for the admin.
- **Real-time Notifications:** Instant Telegram notifications for new comments and newsletter signups.
- **Responsive Design:** Optimized for all screen sizes—from mobile phones to large desktops.
- **Modern UI/UX:** Built using ShadCN UI components with dark and light mode support.
- **SEO Friendly:** Meta tags and structured data for better search engine visibility.



## 📦 Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Set up Environment Variables:**
   Create a `.env` file and add your Firebase and Telegram credentials.

3. **Run the development server:**
   ```bash
   npm run dev
   ```
